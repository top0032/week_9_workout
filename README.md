# 캡스톤 프로젝트 소개 웹 (EC2 + Docker, 또는 로컬)
본 저장소는 캡스톤 프로젝트 단일 소개 페이지와 API를 제공합니다.

## 1. 작품주제(Subject)
- 제목: VocaMate
- 요약: AI를 활용한 영단어 애플리케이션

## 2. 실용적 근거(Rationale)
- 문제: 제한적인 반복 학습의 어려움
- 근거(수치/설문/사례): AI를 활용한 학습 애플리케이션 수요 증가
- 기대 가치: 학습 효율 향상, 다양한 예문 학습

## 3. 핵심기능(Features)
- 기능 1: Gemini API를 활용한 다양한 예문생성
- 기능 2: 단어, 예문을 읽어주는 TTS 기능
- 기능 3: 학습한 단어를 기반으로 한 퀴즈 학습
